# Wireframe - Public Speaking for Kids & Teens Landing Page

## Layout Structure

### Header/Navigation
- Logo program di kiri
- Menu navigation sticky: Home | Program | Benefits | Testimoni | FAQ | Kontak
- CTA Button "Daftar Sekarang" di kanan

### Hero Section (Full Height)
- Background image anak-anak presentasi
- Headline dengan typewriter effect
- Subheadline dengan deskripsi program
- 2 CTA Buttons: "Daftar Sekarang" dan "Coba Gratis"
- Ilustrasi floating elements

### Value Proposition Section
- 4 cards horizontal dengan icon dan deskripsi:
  - Percaya Diri (confidence icon)
  - Komunikasi Jelas (communication icon) 
  - Keberanian Tampil (courage icon)
  - Skill Masa Depan (future skills icon)

### Program Levels Section
- 4 columns grid untuk TK, SD, SMP, SMA
- Each column: age group, focus areas, curriculum highlights
- Hover effects dengan lift animation

### Benefits Section
- Left: Illustration/gambar anak presentasi
- Right: 5 benefit points dengan progress bars
- Animated counters untuk statistics

### Testimonials Section
- Carousel dengan 7 testimoni cards
- Foto anak dan quote orang tua
- Star ratings dan before/after stories

### Learning Methods Section
- 5 method cards dengan icons:
  - Praktik Langsung
  - Fun Learning
  - Roleplay
  - Mini Show
  - Recording Feedback

### Schedule & Pricing Section
- 2 columns layout:
  - Left: Jadwal kelas (weekday/weekend)
  - Right: Pricing tiers dengan hover effects

### FAQ Section
- Accordion layout untuk Q&A
- Smooth expand/collapse animations
- Search functionality untuk FAQ

### Final CTA Section
- Background gradient dengan pattern
- Form registration atau contact form
- Social proof badges

### Footer
- Contact information
- Social media links
- Copyright dan legal info

## Interactive Elements
- Smooth scroll navigation
- Hover effects pada cards dan buttons
- Animated counters dan progress bars
- Testimonial carousel dengan auto-play
- FAQ accordion dengan smooth transitions
- Form validation dengan real-time feedback

## Mobile Considerations
- Responsive grid layouts
- Collapsible navigation menu
- Touch-friendly button sizes
- Optimized image sizes
- Swipe gestures untuk carousel
