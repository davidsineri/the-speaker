# Design Visual Identity - Public Speaking for Kids & Teens

## Design Philosophy

### Color Palette
- **Primary Blue**: #4A90E2 (Professional, Trustworthy)
- **Soft Yellow**: #F5D76E (Cheerful, Energetic) 
- **Gentle Pink**: #F8BBD9 (Warm, Friendly)
- **Neutral Gray**: #F8F9FA (Clean Background)
- **Dark Text**: #2C3E50 (High Contrast)

### Typography
- **Display Font**: Poppins Bold (Modern, Friendly Headers)
- **Body Font**: Inter Regular (Clean, Readable Content)
- **Accent Font**: Fredoka One (Playful Elements)

### Visual Language
- Modern minimalism dengan sentuhan playful
- Ilustrasi flat design yang ramah anak
- Banyak white space untuk readability
- Card-based layout yang organized
- Subtle animations untuk engagement

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth animations and transitions
- **Typed.js**: Typewriter effect untuk hero headline
- **Splide.js**: Testimonial carousel
- **ECharts.js**: Progress visualization
- **p5.js**: Interactive background elements
- **Splitting.js**: Text animation effects

### Animation Effects
- **Hero Typewriter**: Headline muncul huruf per huruf
- **Scroll Reveal**: Konten muncul saat di-scroll
- **Card Hover**: Lift effect dengan shadow
- **Button Pulse**: CTA animation yang eye-catching
- **Progress Bars**: Animated skill indicators

### Header Effect
- Clean background dengan subtle gradient
- Animated shapes menggunakan p5.js
- Floating elements yang responsive
- No overwhelming effects

### Layout Features
- **Sticky Navigation**: Easy access menu
- **Responsive Grid**: Mobile-first design
- **Interactive Cards**: Hover effects
- **Smooth Scrolling**: Enhanced UX
- **Loading Animations**: Professional feel
